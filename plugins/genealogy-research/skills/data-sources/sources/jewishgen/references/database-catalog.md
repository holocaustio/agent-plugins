## Notes

- **UUID df values** are the full identifiers used by the JewishGen API
- **Short codes** are legacy identifiers that also work in the `df` parameter
- Some databases have multiple df values (old vs new versions) — try both if one returns no results
- **This file is read-only** — agents write new discoveries to `.discoveries/database-catalog.md` in the project root
- Agents always check BOTH this file and `.discoveries/database-catalog.md` when looking up databases
- Run `/promote-discoveries` to merge local discoveries back into this file
- Always run a unified search for new surnames to discover databases not listed here
